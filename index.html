Memory Cards

The classic game of memeory cards reimagined for the digital age. Built on HTML, CSS, and Vanilla JavaScript, this card game was one of my first projects. Click a card and then select another; the cards then compare to see if they're a match and flip back over when they aren't a match. Select new game once all cards have been matched to start again.

Crypto Market

Crypto currency confuses and counfounds many people; myself included. Crypto Market is a live updating price tracker on your favorite crypto coins. Crypto Market is powered by CoinGeko API to fetch live price, market cap, and other coin data. The front end is created using React.js and styled with React-Bootstrap, a CSS framework developed by Twitter. Selecting a coin gives you more information on that coin. Select popular coins to display the current top 10 coins according to CoinGeko. Visit the live site below to check it out!

Snapp.r Social

snapp.r social is an image sharing social media platform created by a small dev team of 3. This is a full stack application with the front end created using React.js and Django for the backend. Users will be able to upload images from their device and it will be displayed on the image feed. Images are stored and serviced by AWS S3 to create a permanant cloud database. Styling is created using React-Bootstrap and is fully responsive for mobile and desktop. Future features include: direct messages, Google and Apple user sign in, video support, and custom user profile. Vist the live site to check it out!
